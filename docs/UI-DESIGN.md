# LecTrans - UI 设计规范

## 1. 设计原则

### 1.1 核心原则
- **低调隐蔽**：课堂使用不显眼
- **信息密度**：一屏展示更多内容
- **操作简单**：3 个按钮完成所有操作
- **阅读舒适**：长时间使用不疲劳

### 1.2 设计关键词
```
暗黑 · 简洁 · 高效 · 沉浸
```

```yaml

  # 背景色
  background:
    primary: "#0E1117"      # 主背景（Streamlit 暗黑）
    secondary: "#1A1D24"    # 卡片背景

    primary: "#FAFAFA"      # 主文字
    secondary: "#A3A8B8"    # 次要文字
    muted: "#6B7280"        # 弱化文字
  
  # 强调色
  accent:
    primary: "#4F8CFF"      # 主强调（按钮、链接）
```

## 3. 字体系统

### 3.1 字体选择

```yaml
typography:
  # 韩语字体
  korean:
    primary: "Pretendard"           # 现代韩语字体
    fallback: "Apple SD Gothic Neo", "Malgun Gothic", "sans-serif"

    primary: "Noto Sans SC"         # 简体中文
```

```yaml
font_sizes:
```

## 5. 组件设计

### 5.1 翻译条目组件

```
┌─────────────────────────────────────────────────────┐
│  [10:23:45]                                          │
│  ┌─────────────────────┬─────────────────────┐      │
│  │ 안녕하세요, 오늘    │ 你好，今天我们要    │      │
│  │ 컴퓨터 과학 수업을  │ 开始计算机科学课程  │      │
│  │ 시작하겠습니다.     │                     │      │
│  └─────────────────────┴─────────────────────┘      │
└─────────────────────────────────────────────────────┘

属性：
- timestamp: 时间戳（右上角，灰色小字）
- korean_text: 韩语文本（左侧，白色）
- chinese_text: 中文翻译（右侧，白色）
- 高度：自适应内容
- 间距：16px 底部间距
```

### 5.2 控制按钮组

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  [▶ Start Recording]    [● Connected]    [📝 Summary]  │
│                                                         │
└─────────────────────────────────────────────────────────┘

按钮规格：
- Start/Stop: 
  - 尺寸: 160px × 48px
  - 圆角: 8px
  - 颜色: #4F8CFF (启动) / #FF4B4B (停止)
  - 文字: 白色，16px

- Status:
  - 尺寸: 120px × 32px
  - 圆角: 16px（胶囊）
  - 颜色: #00D97E (已连接) / #FFB020 (连接中) / #FF4B4B (断开)
  - 指示灯: 8px 圆点

- Summary:
  - 尺寸: 120px × 48px
  - 圆角: 8px
  - 颜色: #FFB020
  - 文字: 白色，16px
```

## 6. 交互设计

```
用户打开应用
      │
      ▼
┌─────────────────┐
│  检查 API 配置  │
└────────┬────────┘
         │
         ├──── 已配置 ────▶ 显示主界面
         │
         └──── 未配置 ────▶ 弹出设置面板
                              │
                              ▼
                        ┌─────────────────┐
                        │  用户输入 API   │
                        └────────┬────────┘
                                 │
                                 ▼
                        ┌─────────────────┐
                        │  测试连接      │
                        └────────┬────────┘

                                 ├──── 成功 ────▶ 保存并进入主界面
```

## 8. 动画设计

### 8.1 过渡动画

```css
/* 总结面板滑入 */
.summary-panel {
  transform: translateX(100%);
  transition: transform 300ms ease-out;
}
.summary-panel.active {
  transform: translateX(0);
}

/* 按钮状态变化 */
.btn {
  transition: background-color 200ms ease;
}

/* 新消息出现 */
.message-enter {
  opacity: 0;
  transform: translateY(10px);
  animation: slideIn 200ms ease-out forwards;
}

@keyframes slideIn {
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

### 8.2 加载动画

```
转录中动画：
[● ○ ○ ○]  Processing audio...
```

## 9. 无障碍设计

### 9.1 键盘导航
- `Space`: 开始/停止录音
- `S`: 打开/关闭总结面板
- `Esc`: 关闭弹出面板
- `Tab`: 在控件间切换

### 9.2 屏幕阅读器
- 所有按钮添加 aria-label
- 状态变化添加 aria-live 通知
- 确保颜色对比度 > 4.5:1
