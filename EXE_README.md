# LecTrans Windows EXE 打包说明

## 📦 打包步骤（3 步完成）

### 步骤 1：复制项目到 Windows

将整个 `/root/lectrans/` 文件夹复制到 Windows 电脑

### 步骤 2：运行打包脚本

双击 `build.bat`，等待打包完成（约 2-5 分钟）

### 步骤 3：获取 EXE

打包完成后，`dist\LecTrans.exe` 就是可执行文件

## 🔑 API 获取地址

### 语音识别（Groq）

1. 访问 https://console.groq.com
2. 注册账号
3. 创建 API Key
4. 复制到设置中

### 翻译/总结（小米 MiMo）

1. 访问 https://mimo.xiaomi.com
2. 注册账号
3. 获取 API Key
4. 复制到设置中

## 📝 使用流程

1. **首次运行**：双击 `LecTrans.exe`
2. **配置 API**：点击"设置"，填入 API Key
3. **测试连接**：点击"测试连接"确认配置正确
4. **开始录音**：点击"▶️ 开始录音"
5. **查看翻译**：实时显示韩语原文和中文翻译
6. **生成总结**：点击"📝 生成总结"
7. **保存笔记**：点击"💾 保存"或"📥 导出"

---

## ⚠️ 注意事项

1. **打包环境**：必须在 Windows 上打包，不能跨平台
2. **杀毒软件**：可能被误报，添加白名单即可
3. **文件大小**：打包后约 30-50MB，正常现象
4. **首次运行**：可能需要几秒启动时间

## 🐛 常见问题

### Q: 打包失败？
A: 确保安装了所有依赖：`pip install openai groq httpx pydantic python-dotenv pyinstaller`

### Q: 运行时闪退？
A: 在命令行运行 exe 查看错误信息

### Q: 翻译失败？
A: 检查 API Key 是否正确，网络是否正常

### Q: 如何更新版本？
A: 修改代码后重新运行 `build.bat`
